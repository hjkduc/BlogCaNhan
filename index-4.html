Stream I/O: Dòng chảy dữ liệu trong Java

Trong Java, mọi hoạt động nhập/xuất (I/O) đều dựa trên khái niệm Stream (luồng). Một luồng là một chuỗi dữ liệu tuần tự, có thể đến từ nhiều nguồn khác nhau như file, bàn phím, hoặc một kết nối mạng.

Khi lập trình Socket, chúng ta sử dụng các luồng để gửi và nhận dữ liệu giữa Client và Server.

Các loại Stream chính:

  • InputStream: Luồng dùng để đọc dữ liệu. Phương thức cốt lõi là read().
  • OutputStream: Luồng dùng để ghi dữ liệu. Phương thức cốt lõi là write().

Java cung cấp nhiều lớp "trang trí" (Decorator) để tăng cường chức năng cho các luồng cơ bản. Ví dụ, để gửi văn bản một cách hiệu quả qua Socket, chúng ta thường làm như sau:


// Lấy luồng output cơ bản từ Socket
OutputStream outputStream = socket.getOutputStream();

// Bọc nó bằng một PrintWriter để có thể gửi văn bản theo từng dòng
PrintWriter writer = new PrintWriter(outputStream, true); // true để tự động flush

// Gửi dữ liệu
writer.println("Hello Server!");

Việc kết hợp các lớp Stream một cách khéo léo giúp việc xử lý dữ liệu mạng trở nên đơn giản và mạnh mẽ hơn rất nhiều.
